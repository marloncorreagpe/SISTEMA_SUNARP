============================================================
  PROYECTO UNIVERSITARIO - SUNARP CON BASE DE DATOS SQL
============================================================

Este proyecto es una version academica del seguimiento de titulos
registrales SUNARP. No depende de sistemas privados externos para ser
presentado: usa Python, SQL como base del modelo de datos, menu de consola
y formularios graficos. La ruta recomendada para revision es SQL Server
2022; tambien se deja una ruta alternativa con MySQL/MariaDB desde XAMPP.

OBJETIVO
--------
Digitalizar el control de titulos registrales que antes se revisaban de
forma manual. La base de datos SQL permite guardar, buscar, actualizar,
eliminar y reportar los titulos con trazabilidad.

ARQUITECTURA
------------
1. Interfaz de consola:
   src/main.py

2. Conexion SQL:
   src/database.py
   src/config.py

3. Operaciones CRUD y reportes:
   src/repository.py

4. Base de datos SQL Server 2022:
   sql_sqlserver/01_schema_sqlserver_2022.sql
   sql_sqlserver/02_seed_demo_sqlserver_2022.sql

5. Base de datos alternativa MySQL/MariaDB:
   sql/01_schema.sql
   sql/02_seed_demo.sql
   sql/04_migracion_partida_en_base.sql

INSTALACION
-----------
OPCION A - SQL SERVER 2022

1. Instalar SQL Server 2022, ODBC Driver 18 for SQL Server y sqlcmd.

2. Instalar driver de Python:
   pip install -r requirements.txt

   Tambien puede ejecutar:
   INSTALAR_DEPENDENCIAS.bat

3. Cargar base y datos demo:
   CARGAR_SQLSERVER_2022.bat

4. Abrir formularios:
   FORMULARIOS_SQLSERVER_2022.bat

5. Abrir menu de consola:
   EJECUTAR_MENU_SQLSERVER_2022.bat

OPCION B - MYSQL/MARIADB CON XAMPP

1. Instalar XAMPP y activar MySQL/MariaDB.

2. Crear la base y cargar datos demo desde phpMyAdmin o terminal:
   sql/01_schema.sql
   sql/04_migracion_partida_en_base.sql
   sql/02_seed_demo.sql

   Si XAMPP esta instalado en C:\xampp, tambien puede ejecutar:
   CARGAR_BASE_DATOS.bat

3. Ejecutar:
   python src/main.py

   Tambien puede ejecutar:
   EJECUTAR_MENU.bat

4. Para abrir phpMyAdmin:
   ABRIR_PHPMYADMIN.bat

5. Para generar una evidencia tecnica reproducible:
   python src/demo_profesor.py

CONFIGURACION OPCIONAL
----------------------
Por defecto se conecta a:
  motor: mssql
  servidor SQL Server: localhost\SQLEXPRESS

Para la opcion alternativa MySQL/MariaDB:
  motor: mysql
  host: 127.0.0.1
  puerto: 3306
  usuario: root
  clave: vacia
  base: sunarp_academico

Tambien se puede configurar con variables de entorno:
  SUNARP_DB_ENGINE=mysql o mssql
  SUNARP_DB_HOST
  SUNARP_DB_PORT
  SUNARP_DB_USER
  SUNARP_DB_PASSWORD
  SUNARP_DB_NAME
  SUNARP_SQLSERVER

IMPORTAR RESULTADOS DESDE CSV
-----------------------------
Si se tiene un archivo resultados_full.csv, se puede traspasar a SQL con:

  python src/importar_resultados_csv.py "ruta\al\resultados_full.csv"

ESTADO DE PRUEBA LOCAL
----------------------
La base incluye scripts para SQL Server 2022 y una alternativa probada con
MariaDB 10.4.32 de XAMPP. Se cargaron 50 titulos demo, se verificaron las
vistas de resumen y se ejecuto el menu con busqueda, estadisticas, reporte por
bloque y exportacion CSV.
Tambien se agregaron pruebas automaticas en tests/test_integracion_sql.py.
La evidencia tecnica se genera en salidas/evidencia_demo_profesor.txt.
